# HR-Employee-Attrition-Analysis
End-to-end HR analytics project using Python and Machine Learning to analyze employee attrition, identify key turnover factors, and provide data-driven retention strategies.


Project Overview


Employee attrition is one of the biggest challenges faced by organizations because high turnover increases recruitment costs, reduces productivity, and impacts employee morale.

This project analyzes employee data to identify the key factors influencing attrition and develops a machine learning model to predict employees who are at risk of leaving.


Business Problem

Organizations often struggle to understand why employees leave. This project aims to identify the drivers of attrition and provide actionable recommendations to improve employee retention.



Objectives

• Clean and preprocess HR data

• Perform Exploratory Data Analysis (EDA)

• Identify factors contributing to employee attrition

• Build predictive machine learning models

• Generate business recommendations for HR teams


Dataset Information

Source:

IBM HR Analytics Employee Attrition Dataset

Records:

1470 Employees

Target Variable:

Attrition (Yes/No)


Tools Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Scikit-Learn

Jupyter Notebook (Kaggle)


Key Findings


✔ Employees working overtime showed higher attrition rates.

✔ Employees with lower monthly income were more likely to leave.

✔ Job satisfaction influenced employee retention.

✔ Years at the company affected attrition probability.

✔ Machine learning successfully predicted attrition risk.




Business Recommendations

• Monitor employees with frequent overtime.

• Improve career growth opportunities.

• Review compensation for high-risk employee groups.

• Increase engagement programs for new employees.

• Use predictive analytics to identify employees at risk of leaving.
